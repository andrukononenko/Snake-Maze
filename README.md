🐍 Snake Maze – Multi‑Language Edition
A Snake game where you navigate a snake through a procedurally generated maze!
Collect food, avoid walls, and grow longer – but the maze adds a new layer of challenge.
Built in 7 programming languages – each implementation features maze generation, snake movement, scoring, and real‑time keyboard controls.

✨ Features
Procedural maze generation – each game starts with a unique maze (DFS algorithm).

Snake movement – smooth, grid‑based movement with arrow keys or WASD.

Food spawning – appears in a random empty cell (not on walls or snake).

Score & speed – score increases with each food; speed gradually rises.

Level progression – after a certain score, a new maze is generated (optional).

Wall collision – snake dies if it hits a wall.

Self‑collision – snake dies if it hits itself.

Restart – press R to restart.

Cross‑platform – works on any terminal (Windows, macOS, Linux).

🚀 How to Run
Each file is standalone – run it with the appropriate interpreter/compiler.

Language	Command
Python	python snake.py
Go	go run snake.go
JavaScript	node snake.js
C#	dotnet run (or csc Snake.cs && Snake.exe)
Java	javac Snake.java && java Snake
Ruby	ruby snake.rb
Swift	swift snake.swift
🎮 Controls
Arrow keys or WASD – move the snake.

R – restart after game over.

Q / Esc – quit.

📦 Maze Generation
Uses Depth‑First Search (DFS) to carve paths through a grid.

Walls are represented by #, paths by spaces.

The maze is regenerated after each level or on restart.
